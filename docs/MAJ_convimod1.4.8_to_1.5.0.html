Bienvenue sur le fichier de mise à jour de la preMOD :

ConviMod V1.4.8 vers ConviMod V1.5.0

Avant propos :

Vous ne pouvez pas utiliser cette preMOD comme base pour commencer et/ou distribuer votre propre preMOD. Vous ne pouvez pas réutiliser les scripts non-distribués présents dans cette preMOD dans une autre preMOD ou sur votre forum personnel :
  • soit ils sont en cours de développement,
  • soit ils sont exclusifs à la preMOD ConviMod.
Il est demandé que les copyrights de la preMOD (nom, lien vers le site EzCom-fr et numéro de version) visibles en bas de page soient conservés, par respect pour le travail effectué pour vous offrir une preMOD de qualité. Dans le cas où ceux-ci seraient supprimés ou tronqués, aucun support ne sera effectué.

MAJ :

Pour mettre à jour votre preMod ConviMod V1.4.8 vers la V1.5.0 :
  • Connectez vous en tant qu'admin à votre forum.
  • Sur votre FTP, videz le dossier cache
  • Uploadez le dossier :
    • install
  • puis lancer le script suivant depuis votre navigateur :
    • install.php
  • de la façon suivante :
    • http://lien_de_votre_forum/install/install.php
      En choississant l'option Mis à jour
  • supprimez tout les fichiers et dossiers de votre ftp sauf :
    • cache
    • files
    • config.php
  • uploadez tous les dossiers de l'archive sauf :
    • cache
    • files
    • config.php

Si vous avez une preMOD ConviMod 1.3.5 à 1.4.6
  • Utilisez le script Clean Table pour retourner vers un phpBB2 vanilla (de base) puis reportez-vous au fichier :
    • MAJ_phpBB_to_convimod_1.5.0.html

Un petit rappel :

01. CHMODs à effectuer :

    Dossier Root ( Racine du ftp )
      cache : CHMOD 777
      efiles : CHMOD 777
      files : CHMOD 777
      includes
        def_bbc_box.php : CHMOD 666

02. Liste des MODs installés :

    Vous trouverez l'ensemble des MODs installés sur la preMOD en vous rendant à cette adresse : ConviMod

    Nous rappelons que les MODs installés sur cette preMOD ne proviennent pas de nous.
    Les Copyrights ont donc été laissés, ce qui est tout à fait normal.
    Les développeurs des MODs et autres Add-Ons restent les détenteurs de ces MODs.

03. Les mises à jour de la preMOD :

    Il se peut que vous installiez des MODs supplémentaires.
    Ayez toujours dans un répertoire adapté sur votre système l'ensemble des MODs et autres modifications que vous apportez à la preMOD.
    En effet, lors des mises à jour ( phpBB, MODs, ... ), nous ne mettrons à jour que les fichiers d'origine.

04. Détenteurs de forum phpBB Vanilla ( Forum de base ) :

    Une mise à jour, sous forme d'un fichier SQL, va bientôt sortir également.
    Elle vous permettra d'actualiser votre base de données afin d'être compatible avec la preMOD ConviMod 1.5.x ...

05. Support de la preMOD :

    Toute demande de support ne se fera que sur EzCom-Fr, et pas ailleurs.
    Les règles du phpBB Group sont ainsi, et elles ne doivent pas être changées.

    Nous rappelons néanmoins que, comme sur la preMOD 1.1.x, 1.2.x, 1.3.x et 1.4.x, nous ne recommandons pas les hébergeurs comme Free, ou tout autre hébergeur ne supportant les CHMODs.
